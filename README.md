# AI Profile Website
This is a modern AI-themed portfolio website for a B.Tech Artificial Intelligence & Machine Learning student. It features a dynamic loader, typing effects, particle animations, interactive project cards, and detailed sections for skills, education, projects, and achievements.

# Portfolio
Welcome to my Portfolio Website!
This website showcases my education, technical skills, projects, certifications, internships, and achievements.
🌐 https://lekhasree0910.github.io/me/

## Features
- **AI-Themed Design**: High-tech interface with gold accents.
- **Dynamic Loader**: Animated boot sequence with typing effect.
- **Interactive Elements**: Custom cursor, floating particles, and 3D-tilt project cards.
- **Typed Text**: Auto-typing for the hero section headline.
- **Responsive Layout**: Optimized for both desktop and mobile devices.

## Technologies Used
- **HTML5**: Structure and semantics.
- **CSS3**: Styling, animations, and glassmorphism effects.
- **JavaScript (Vanilla)**: DOM manipulation, event handling, and GSAP animations.
- **GSAP (GreenSock)**: Advanced animations and scroll effects.
- **Typed.js**: Typewriter effect for the hero section.

## Structure
The website is organized into the following main sections:

- **Loader**: A full-screen overlay that animates on page load.
- **Hero Section**: Introduces the candidate with a headline, typing effect, and call-to-action buttons.
- **About Section**: Displays candidate details and statistics.
- **Skills Section**: Showcases technical skills organized by category.
- **Projects Section**: Detailed cards for academic and personal projects.
- **Timeline (Education & Experience)**: Chronological history of education and internships.
- **Certifications**: A grid of obtained certifications.
- **Events**: List of workshops and conferences attended.
- **Leadership**: Positions held and leadership roles.
- **Socials**: Links to GitHub, LinkedIn, and other platforms.

## How to Run
1.  **Clone the repository** or download the source code.
2.  **Open `index.html`** in any modern web browser.
